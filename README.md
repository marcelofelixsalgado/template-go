### Preconditions

Requires at least the following executables on `$PATH` to run succesfully:
- Git
- Go >= 1.15

## Usage

To execute from source:
```
go run
```

To execute the unit tests:
```
go test -v ./...
```

To build a local executable:
```
go build
```

To build a global executable (bin folder):
```
go install
```
